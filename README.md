# Alto Cali Hotel Boutique

Proyecto premium desarrollado con Next.js App Router, React, Tailwind CSS, Framer Motion y Lucide React.

## Ejecutar localmente

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Estructura

- `app/`: rutas principales.
- `components/`: componentes reutilizables.
- `sections/`: secciones de la home.
- `data/`: contenido editable.
- `lib/`: utilidades.
- `styles/`: carpeta reservada para estilos futuros.
